# Currency column for Yii

By Alexey Samoylov (<alexey.samoylov@gmail.com>).

## Requirements

- **PHP 5.4**
- **Yii 1.x**

### Examples

Usage example:

```php
'columns' => [
	[
		'class' => 'vendor.lagman.yii-currency-column.CurrencyColumn',
	 	'name' => 'amount',
		'currency' => 'USD'
	]	
]
```
